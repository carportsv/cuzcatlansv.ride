# Cuzcatlansv.ride - GitHub Pages

Esta es la versión de producción de la aplicación taxi para GitHub Pages.

## 🔒 Seguridad

- ✅ **Sin claves sensibles expuestas**
- ✅ **Solo claves públicas/anónimas**
- ✅ **Configuración segura para producción**

## 🚀 Despliegue

1. Sube este directorio a tu repositorio de GitHub
2. Configura GitHub Pages en la rama `main` o `gh-pages`
3. La aplicación estará disponible en: `https://tu-usuario.github.io/tu-repo/`

## 📁 Estructura

- `index.html` - Aplicación principal
- `js/` - Scripts de la aplicación
- `css/` - Estilos
- `assets/` - Imágenes y recursos

## 🔧 Configuración

Las variables de entorno están hardcodeadas en `index.html` y `js/config.js` para funcionar en GitHub Pages.

**¡No modifiques las claves!** Son las claves públicas/anónimas que son seguras para el cliente.
